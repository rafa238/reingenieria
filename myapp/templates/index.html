Instrucciones para acceder al crud generado por el framework django

  1. Hay que dirigirse a la ruta "/admin"
  2. En el formulario de usuario y contraseña se ingresara el usuario: admin y contraseña:admin
  3. Para acceder al crud de los modelos hay un apartado llamado CRUD donde se podra hacer todas las accion CRUD de los modelos de la base de datos generado por django

Toda la interfaz grafica fue generada por el framework django, pues una de las fortalezas de django es que genera automaticamente una interfaz de admin a partir de los modelos que se creen en su ORM e incluso modulos por default que django crea, como el inicio de sesion de admin que posee. Estas caracteristicas hacen fácil y rapido la codificación y desarrollo de una aplicación web.
